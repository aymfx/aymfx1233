两者的不同

WeakSet 对象中只能存放对象引用, 不能存放值, 而 Set 对象都可以.

WeakSet 对象中存储的对象值都是被弱引用的, 如果没有其他的变量或属性引用这个对象值, 则这个对象值会被当成垃圾回收掉. 正因为这样, WeakSet 对象是无法被枚举的, 没有办法拿到它包含的所有元素

WeakSet 不可迭代 于是 for of foreach keys() values() 方法都没有
