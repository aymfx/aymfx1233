性能优化

  • 使用生产环境下的react.js,这样可以减少体积以及在生产环境下不需要的提示
  • 创建 reactAPP —> npm run build 进行打包压缩
  • 利用branch创建更高效的生产版本
    • npm install –save-dev uglify-js-brunch 或者 yarn add –dev uglify-js-brunch
    • brunch build -p
  • 利用webpack创建，如果是用create-react创建的可以不考虑重新建
